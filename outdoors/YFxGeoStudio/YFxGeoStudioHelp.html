  1. 打开 YFxGeoStudio.html。手机测试新版本时，优先从首页的“YFxGeoStudio 最新版”进入。
  2. 点击 🛤️ 打开经典轨迹目录，或点击 📂 导入本地 KML / 离线缓存包。
  3. 轨迹加载后，在地图上拖动、双指缩放；点击轨迹列表或地图上的轨迹查看距离、耗时、上升、下降和海拔信息。

3. 选择经典轨迹

  1. 点击 🛤️ 打开经典轨迹目录。
  2. 用搜索框输入地点、山名或文件名，例如“武功山”“太阳山”。
  3. 点击普通 KML 会直接载入轨迹；点击 .yfxcache.zip 会先下载离线包，再导入轨迹和瓦片。
  4. 离线缓存包导入完成后，浏览器通常会同时请求保存到手机或电脑下载目录。若手机自动保存被拦截，可在弹窗中再点“保存离线包”。
经典轨迹列表来自线上目录。若刚上传新文件，GitHub Pages 可能需要短时间部署；列表或文件短暂 404 时，稍后刷新再试。

4. 查看轨迹和标记点

  1. 点击左下角轨迹列表中的轨迹，或直接点击地图上的轨迹线。
  2. 信息窗口会显示轨迹距离、耗时、累计上升、累计下降、海拔范围和高度变化图。
  3. 双击轨迹列表中的轨迹，可飞到对应路线并高亮。
  4. 点击标记点可查看名称、经纬度、海拔和时间；手机上长按标记点可打开导航操作。

5. 定位、朝向和轨迹记录

  1. 点击 📍 请求当前位置权限，地图会移动到当前位置。
  2. 如果手机允许方向传感器，定位箭头会尽量按手机朝向显示；未授权时使用 GPS 行进方向。
  3. 双击 📍 开始 GPS 轨迹记录，按钮会变成记录状态。
  4. 再次点击记录状态按钮结束记录，输入名称后保存到当前工作区，并可导出 KML。
手机浏览器可能分别询问定位权限和方向权限。若拒绝权限，定位或朝向功能会降级，但普通轨迹浏览仍可使用。

6. 手绘、编辑和导出

  1. 点击 ✏️ 进入手绘模式，在地图上连续点选生成轨迹点。
  2. 手绘完成后保存轨迹名称。手绘轨迹点可拖动调整；导入的轨迹默认不可拖动，避免误改原始数据。
  3. 编辑轨迹或标记点时，可修改名称、备注、颜色、图标、海拔和时间。
  4. 点击 💾 导出当前数据；可导出合并后的 KML，用于其它地图软件或设备。

手机没有鼠标右键。工具中多数编辑动作可通过点击、双击或长按触发；如果某个操作不方便，建议横屏或在电脑端完成精细编辑。

7. 离线缓存和 YFxDataCache

  1. 点击帮助窗口中的 🗺️ 可打开 YFxDataCache。
  2. 在 YFxDataCache 中加载 KML 或经典轨迹，点击开始缓存，浏览需要的图层和区域。
  3. 鸟瞰图中黑色表示未浏览，黄色表示浏览过但缓存失败，绿色表示成功写入离线包。
  4. 保存后得到 .yfxcache.zip，之后可在 YFxGeoStudio 中用 📂 导入。
浏览器本身会缓存部分已看过的地图瓦片，但不保证完整。需要稳定离线检查时，应使用 YFxDataCache 导出 .yfxcache.zip。

8. 手机端常见问题

总是打开旧版本

从本站首页点击 YFxGeoStudio 最新版 进入。这个入口会清理 YFxGeoStudio 应用外壳缓存并追加时间戳参数。若仍旧异常，可关闭浏览器标签页后重新进入。

地图无法缩放页面

地图会优先接管触摸手势。需要缩放整个网页时，点击 🔒 锁定地图触控，再用浏览器手势缩放页面；完成后再次点击解锁。

不能导入文件

手机文件选择器可能按文件类型过滤。请确认文件扩展名是 .kml、.zip 或 .yfxcache.zip；若文件来自微信或网盘，先保存到系统“文件/下载”目录再导入。

离线包没有自动保存

电脑端 Chrome 通常会自动保存；手机端经常需要用户确认。看到“保存离线包”按钮时，可以再次点击触发保存。

定位不准或没有方向

请确认浏览器允许定位权限，并尽量在室外空旷位置等待 GPS 稳定。方向传感器权限被拒绝时，工具会改用 GPS 行进方向。

9. 桌面端补充

电脑端支持鼠标右键、多选列表、拖拽编辑等更精细操作。旧版桌面截图仍保留在 help_pic/ 目录中，后续可按新版手机界面重新生成截图替换。
