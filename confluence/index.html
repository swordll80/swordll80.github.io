confluence

在主页点击confluence节点，打开了 swordll80.github.io/confluence.html

confluence.html是运行在本地的web程序，需要获得swordll80.github.io\confluence\目录的读写权。

从confluence.html返回主页时，会提醒要执行 swordll80.github.io\confluence\git.bat 提交本地已经修改的html到github。

git.bat由confluence.html自动生成，

git.bat需要人工填写 set "GITHUB_TOKEN=***************************************" 中的星号部分

双击运行git.bat后，git会弹窗询问github用户名和密码

一般只有第一次运行时需要输入用户名密码。
