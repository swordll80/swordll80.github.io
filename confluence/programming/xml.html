一、XML 基础语法
1XML 声明<?xml version="1.0" encoding="UTF-8" standalone="yes"?> <!-- 必须放在第一行, 之前不能有空格 -->XML 1.0
2元素与标签<book id="1"> <title>XML入门</title> <price>29.99</price> </book> <!-- 标签区分大小写, 必须有闭合 -->XML 1.0
3属性<person id="p001" name="Alice" age="25"/> <!-- 属性值必须加引号(单双均可) --> <!-- 同一元素中属性名不能重复 -->XML 1.0
4注释<!-- 这是注释 --> <!-- 注释不能嵌套, 不能出现在声明前 --> <!-- 注释内不能包含 -- -->XML 1.0
5CDATA 节<code><![CDATA[ if (a < b && c > d) { return true; } ]]></code> <!-- CDATA 内的 < > & 等不解析 -->XML 1.0
6实体引用<text>5 &lt; 10 &amp; price &gt; 0</text> <!-- 预定义实体: &lt; &gt; &amp; &quot; &apos; -->XML 1.0
7处理指令 (PI)<?xml-stylesheet type="text/xsl" href="style.xsl"?> <!-- 给应用程序传递信息的指令 -->XML 1.0
8命名空间<root xmlns="http://example.com/ns" xmlns:html="http://www.w3.org/1999/xhtml"> <element/> <html:table>...</html:table> </root>XML 1.0
9格式良好 (Well-formed)<!-- 1. 有且只有一个根元素 --> <!-- 2. 所有标签正确闭合 --> <!-- 3. 标签正确嵌套 --> <!-- 4. 属性值加引号 --> <!-- 5. 实体引用正确 --> <!-- 满足以上即为 Well-formed XML -->XML 1.0
二、DTD (文档类型定义)
10内部 DTD<?xml version="1.0"?> <!DOCTYPE note [ <!ELEMENT note (to,from,heading,body)> <!ELEMENT to (#PCDATA)> <!ELEMENT from (#PCDATA)> <!ATTLIST note id ID #REQUIRED> ]> <note id="n1"><to>Alice</to>...</note>XML 1.0
11外部 DTD<!-- 私有: --> <!DOCTYPE note SYSTEM "note.dtd"> <!-- 公共: --> <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">XML 1.0
12DTD 元素声明<!-- (#PCDATA): 可解析字符数据 --> <!ELEMENT name (#PCDATA)> <!-- EMPTY: 空元素 --> <!ELEMENT br EMPTY> <!-- ANY: 任意内容 --> <!ELEMENT data ANY> <!-- 子元素序列: --> <!ELEMENT person (name, age?, address*)> <!-- 选择: --> <!ELEMENT fruit (apple | orange | banana)>XML 1.0
13DTD 出现次数指示符<!-- +: 一次或多次 --> <!-- *: 零次或多次 --> <!-- ?: 零次或一次 --> <!-- 无符号: 恰好一次 -->XML 1.0
14DTD 属性声明<!ATTLIST element attr-name attr-type default-value> <!-- 类型: CDATA, ID, IDREF, IDREFS, NMTOKEN, ENTITY --> <!-- 默认值: #REQUIRED, #IMPLIED, #FIXED "val", "默认值" -->XML 1.0
三、XML Schema (XSD)
15Schema 声明<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema"> <!-- 定义在此 --> </xs:schema>XSD 1.0
16元素定义<xs:element name="age" type="xs:integer" default="18"/> <xs:element name="note"> <xs:complexType> <xs:sequence> <xs:element name="to" type="xs:string"/> </xs:sequence> </xs:complexType> </xs:element>XSD 1.0
17内置数据类型<!-- 字符串: xs:string, xs:token, xs:normalizedString --> <!-- 数值: xs:integer, xs:int, xs:decimal, xs:float, xs:double --> <!-- 日期: xs:date, xs:time, xs:dateTime --> <!-- 其他: xs:boolean, xs:anyURI, xs:ID, xs:IDREF -->XSD 1.0
18约束 (restriction)<xs:simpleType name="ageType"> <xs:restriction base="xs:integer"> <xs:minInclusive value="0"/> <xs:maxInclusive value="150"/> </xs:restriction> </xs:simpleType>XSD 1.0
19Schema vs DTD<!-- Schema: 支持数据类型, 命名空间, 扩展性强 --> <!-- DTD: 简单, 但不支持类型, 不支持命名空间 --> <!-- Schema 本身也是 XML, DTD 不是 -->-
四、XPath
20路径表达式// /bookstore/book[1] → 第一本书 // /bookstore/book[last()] → 最后一本 // /bookstore/book[price>35] → 价格>35的书 // /bookstore/book/title | //price → 并集 // //title[@lang='en'] → 属性 lang='en' 的 title // /bookstore/book[position()<3] → 前两本XPath 1.0
21XPath 轴 (Axes)// ancestor::* → 所有祖先 // child::* → 所有子元素 // descendant::* → 所有后代 // following::* → 之后的所有节点 // preceding::* → 之前的所有节点 // parent::* → 父节点 // attribute::* → 所有属性XPath 1.0
22XPath 函数// count(//book) → 节点数 // string(/book/title) → 字符串值 // concat('a','b') → 字符串拼接 // contains(title, 'XML') → 包含判断 // starts-with(name, 'A') → 前缀判断 // substring(str, 1, 3) → 子串 // sum(//price) → 求和XPath 1.0
五、XSLT
23XSLT 基础模板<?xml version="1.0"?> <xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform"> <xsl:template match="/"> <html><body> <h2><xsl:value-of select="//title"/></h2> </body></html> </xsl:template> </xsl:stylesheet>XSLT 1.0
24常用 XSLT 元素<!-- xsl:value-of → 输出节点值 --> <!-- xsl:for-each → 循环 --> <!-- xsl:if → 条件 --> <!-- xsl:choose/when/otherwise → 多条件 --> <!-- xsl:sort → 排序 --> <!-- xsl:apply-templates → 应用模板 -->XSLT 1.0
六、XML 解析方式
25DOM 解析<!-- 将 XML 整个加载到内存, 构建树结构 --> <!-- 优点: 可随机访问任意节点, 可修改 --> <!-- 缺点: 内存占用大, 不适合大文件 --> <!-- API: Document, Element, Node, NodeList -->-
26SAX 解析<!-- 事件驱动, 顺序读取, 不建树 --> <!-- 优点: 内存占用小, 速度快 --> <!-- 缺点: 只能向前读, 不能修改 --> <!-- 回调: startElement, endElement, characters -->-
27StAX (拉式解析)<!-- 类似 SAX 但由程序主动拉取事件 --> <!-- 优点: 比 SAX 更易控制 --> <!-- Java: XMLStreamReader, XMLEventReader -->-

八、常见Bug与陷阱
35特殊字符未转义<!-- < 必须写成 &lt; --> <!-- & 必须写成 &amp; --> <!-- 或用 CDATA 包裹 -->XML 1.0
36编码不匹配<!-- 声明 UTF-8 但文件实际是 GBK → 解析错误 --> <!-- 始终用 UTF-8, 并确保声明与实际一致 -->XML 1.0
37BOM 头问题<!-- UTF-8 BOM (EF BB BF) 在声明前会导致解析失败 --> <!-- 用不带 BOM 的 UTF-8 编码 -->XML 1.0
38空白敏感<!-- XML 保留空白字符, 不像 HTML --> <!-- 格式化时可能引入意外的换行和空格 -->XML 1.0
39命名空间前缀未声明<!-- 使用 ns:tag 前必须 xmlns:ns="uri" -->XML 1.0
