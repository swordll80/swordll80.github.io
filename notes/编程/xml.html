XML 知识手册

44 条
1 XML 声明 XML 1.0
<?xml version="1.0" encoding="UTF-8" standalone="yes"?> <!-- 必须放在第一行, 之前不能有空格 -->
2 元素与标签 XML 1.0
<book id="1"> <title>XML入门</title> <price>29.99</price> </book> <!-- 标签区分大小写, 必须有闭合 -->
3 属性 XML 1.0
<person id="p001" name="Alice" age="25"/> <!-- 属性值必须加引号(单双均可) --> <!-- 同一元素中属性名不能重复 -->
4 注释 XML 1.0
<!-- 这是注释 --> <!-- 注释不能嵌套, 不能出现在声明前 --> <!-- 注释内不能包含 -- -->
5 CDATA 节 XML 1.0
<code><![CDATA[ if (a < b && c > d) { return true; } ]]></code> <!-- CDATA 内的 < > & 等不解析 -->
6 实体引用 XML 1.0
<text>5 &lt; 10 &amp; price &gt; 0</text> <!-- 预定义实体: &lt; &gt; &amp; &quot; &apos; -->
7 处理指令 (PI) XML 1.0
<?xml-stylesheet type="text/xsl" href="style.xsl"?> <!-- 给应用程序传递信息的指令 -->
8 命名空间 XML 1.0
<root xmlns="http://example.com/ns" xmlns:html="http://www.w3.org/1999/xhtml"> <element/> <html:table>...</html:table> </root>
9 格式良好 (Well-formed) XML 1.0
<!-- 1. 有且只有一个根元素 --> <!-- 2. 所有标签正确闭合 --> <!-- 3. 标签正确嵌套 --> <!-- 4. 属性值加引号 --> <!-- 5. 实体引用正确 --> <!-- 满足以上即为 Well-formed XML -->
10 内部 DTD XML 1.0
<?xml version="1.0"?> <!DOCTYPE note [ <!ELEMENT note (to,from,heading,body)> <!ELEMENT to (#PCDATA)> <!ELEMENT from (#PCDATA)> <!ATTLIST note id ID #REQUIRED> ]> <note id="n1"><to>Alice</to>...</note>
11 外部 DTD XML 1.0
<!-- 私有: --> <!DOCTYPE note SYSTEM "note.dtd"> <!-- 公共: --> <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
12 DTD 元素声明 XML 1.0
<!-- (#PCDATA): 可解析字符数据 --> <!ELEMENT name (#PCDATA)> <!-- EMPTY: 空元素 --> <!ELEMENT br EMPTY> <!-- ANY: 任意内容 --> <!ELEMENT data ANY> <!-- 子元素序列: --> <!ELEMENT person (name, age?, address*)> <!-- 选择: --> <!ELEMENT fruit (apple | orange | banana)>
13 DTD 出现次数指示符 XML 1.0
<!-- +: 一次或多次 --> <!-- *: 零次或多次 --> <!-- ?: 零次或一次 --> <!-- 无符号: 恰好一次 -->
14 DTD 属性声明 XML 1.0
<!ATTLIST element attr-name attr-type default-value> <!-- 类型: CDATA, ID, IDREF, IDREFS, NMTOKEN, ENTITY --> <!-- 默认值: #REQUIRED, #IMPLIED, #FIXED "val", "默认值" -->
15 Schema 声明 XSD 1.0
<xs:schema xmlns:xs="http://www.w3.org/2001/XMLSchema"> <!-- 定义在此 --> </xs:schema>
16 元素定义 XSD 1.0
<xs:element name="age" type="xs:integer" default="18"/> <xs:element name="note"> <xs:complexType> <xs:sequence> <xs:element name="to" type="xs:string"/> </xs:sequence> </xs:complexType> </xs:element>
17 内置数据类型 XSD 1.0
<!-- 字符串: xs:string, xs:token, xs:normalizedString --> <!-- 数值: xs:integer, xs:int, xs:decimal, xs:float, xs:double --> <!-- 日期: xs:date, xs:time, xs:dateTime --> <!-- 其他: xs:boolean, xs:anyURI, xs:ID, xs:IDREF -->
18 约束 (restriction) XSD 1.0
<xs:simpleType name="ageType"> <xs:restriction base="xs:integer"> <xs:minInclusive value="0"/> <xs:maxInclusive value="150"/> </xs:restriction> </xs:simpleType>
19 Schema vs DTD -
<!-- Schema: 支持数据类型, 命名空间, 扩展性强 --> <!-- DTD: 简单, 但不支持类型, 不支持命名空间 --> <!-- Schema 本身也是 XML, DTD 不是 -->
20 路径表达式 XPath 1.0
// /bookstore/book[1] → 第一本书 // /bookstore/book[last()] → 最后一本 // /bookstore/book[price>35] → 价格>35的书 // /bookstore/book/title | //price → 并集 // //title[@lang='en'] → 属性 lang='en' 的 title // /bookstore/book[position()<3] → 前两本
21 XPath 轴 (Axes) XPath 1.0
// ancestor::* → 所有祖先 // child::* → 所有子元素 // descendant::* → 所有后代 // following::* → 之后的所有节点 // preceding::* → 之前的所有节点 // parent::* → 父节点 // attribute::* → 所有属性
22 XPath 函数 XPath 1.0
// count(//book) → 节点数 // string(/book/title) → 字符串值 // concat('a','b') → 字符串拼接 // contains(title, 'XML') → 包含判断 // starts-with(name, 'A') → 前缀判断 // substring(str, 1, 3) → 子串 // sum(//price) → 求和
23 XSLT 基础模板 XSLT 1.0
<?xml version="1.0"?> <xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform"> <xsl:template match="/"> <html><body> <h2><xsl:value-of select="//title"/></h2> </body></html> </xsl:template> </xsl:stylesheet>
24 常用 XSLT 元素 XSLT 1.0
<!-- xsl:value-of → 输出节点值 --> <!-- xsl:for-each → 循环 --> <!-- xsl:if → 条件 --> <!-- xsl:choose/when/otherwise → 多条件 --> <!-- xsl:sort → 排序 --> <!-- xsl:apply-templates → 应用模板 -->
25 DOM 解析 -
<!-- 将 XML 整个加载到内存, 构建树结构 --> <!-- 优点: 可随机访问任意节点, 可修改 --> <!-- 缺点: 内存占用大, 不适合大文件 --> <!-- API: Document, Element, Node, NodeList -->
26 SAX 解析 -
<!-- 事件驱动, 顺序读取, 不建树 --> <!-- 优点: 内存占用小, 速度快 --> <!-- 缺点: 只能向前读, 不能修改 --> <!-- 回调: startElement, endElement, characters -->
27 StAX (拉式解析) -
<!-- 类似 SAX 但由程序主动拉取事件 --> <!-- 优点: 比 SAX 更易控制 --> <!-- Java: XMLStreamReader, XMLEventReader -->
28 XHTML -
<!-- HTML 的 XML 化版本, 严格符合 XML 规则 -->
29 SVG -
<!-- 可缩放矢量图形, 基于 XML -->
30 RSS / Atom -
<!-- 内容聚合格式, 基于 XML -->
31 SOAP -
<!-- Web Service 协议, 基于 XML -->
32 AndroidManifest -
<!-- Android 应用配置文件, 基于 XML -->
33 pom.xml (Maven) -
<!-- Java 项目构建配置, 基于 XML -->
34 web.xml -
<!-- Java Web 应用部署描述符 -->
35 特殊字符未转义 XML 1.0
<!-- < 必须写成 &lt; --> <!-- & 必须写成 &amp; --> <!-- 或用 CDATA 包裹 -->
36 编码不匹配 XML 1.0
<!-- 声明 UTF-8 但文件实际是 GBK → 解析错误 --> <!-- 始终用 UTF-8, 并确保声明与实际一致 -->
37 BOM 头问题 XML 1.0
<!-- UTF-8 BOM (EF BB BF) 在声明前会导致解析失败 --> <!-- 用不带 BOM 的 UTF-8 编码 -->
38 空白敏感 XML 1.0
<!-- XML 保留空白字符, 不像 HTML --> <!-- 格式化时可能引入意外的换行和空格 -->
39 命名空间前缀未声明 XML 1.0
<!-- 使用 ns:tag 前必须 xmlns:ns="uri" -->
