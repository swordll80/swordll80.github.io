- 将所有子页面统一为 **左侧两级树导航 + 右侧内容区（iframe嵌入）** 的布局
- 风格与主页 `index.html` 和 `tools/index.html` 保持一致：毛玻璃 header、径向渐变背景、圆角卡片布局
- 每个页面 header 左侧有 `← 返回主页` 链接（使用 `class="back"` 样式），右侧为页面标题

### 设计规范
- CSS 变量统一使用 `--bg`, `--panel`, `--text`, `--muted`, `--line`, `--accent`, `--accent-soft`
- 所有子页面引用 `../styles.css`（或 `../../styles.css`）作为共享基础样式
- `styles.css` 精简为基础变量、链接、`.back`、标题等公用样式
- 页面特有的布局 CSS（header/layout/sidebar/tree/iframe）内嵌在各页面 `<style>` 中
- 树导航使用 `data-toggle` 展开/折叠分类，`data-tool` 触发右侧内容加载
